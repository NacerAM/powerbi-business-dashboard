# Business Performance Dashboard (Power BI)

## Objective
This project demonstrates the creation of a business intelligence dashboard for a small or medium-sized business.

## Tools
- Power BI
- Excel / CSV

## Description
The dashboard presents key performance indicators (KPIs) including revenue, profit, margin, and monthly performance trends.

## Dataset
The dataset used in this project is fictional and created for demonstration purposes.

## Output
Dashboard screenshots are available in the screenshots folder.
